Projets académiques majeurs

Architecture de Fournisseur d'Accès Internet Multi-AS

Conception et déploiement d'une infrastructure Internet complète avec multiples Systèmes Autonomes. Implémentation du routage dynamique inter-domaines (BGP) et intra-domaines (OSPF), segmentation réseau par VLANs, haute disponibilité via Spanning Tree Protocol, déploiement de services réseau (DNS, DHCP, serveurs Web), sécurisation par pare-feu iptables.

Technologies : Linux (FRRouting), OSPF, BGP, VLANs, STP, DNS (bind9), DHCP, iptables

Équipe : 5 personnes (méthodologie Agile)

Chaîne de Transmission Numérique Complète

Modélisation sous MATLAB d'une chaîne de transmission en bande de base. Simulation de canal multi-trajets avec bruit AWGN, implémentation d'algorithmes d'égalisation temporelle (Zero-Forcing, MMSE) et fréquentielle (SC-FDMA), comparaison des performances en fonction du rapport signal/bruit (SNR), optimisation du Taux d'Erreur Binaire (TEB).

Technologies : MATLAB, traitement du signal, modulations numériques

Solveur de Sudoku par Bases de Gröbner

Programme Python pour résoudre des grilles de Sudoku en utilisant une approche algébrique innovante. Modélisation du problème comme un système d'équations polynomiales à 729 variables booléennes (9x9x9), résolution exacte par calcul de bases de Gröbner garantissant l'unicité de la solution. Interface graphique développée avec Tkinter permettant la saisie interactive, la visualisation de la résolution, et l'import/export de grilles.

Technologies : Python 3, Tkinter (interface graphique), SymPy (algèbre symbolique), NumPy (structures de données)

Compétences développées : Modélisation mathématique, algèbre abstraite appliquée, développement d'interfaces graphiques, optimisation algorithmique

Portfolio Professionnel et Projets Web

Développement d'un portfolio en ligne professionnel (site actuel) et réalisation de plusieurs projets d'apprentissage en développement web front-end et back-end. Maîtrise progressive des technologies web modernes à travers des projets concrets : landing page avec animations parallax et effets glassmorphism, calculatrice scientifique interactive avec fonctions avancées, gestionnaire de tâches avec persistance localStorage, application météo avec intégration API OpenWeatherMap.

Technologies : HTML5 (structure sémantique, accessibilité), CSS3 (Flexbox, Grid, animations, responsive design), JavaScript ES6+ (DOM manipulation, Fetch API, async/await), Git (versioning)
